# EduPlatform — Frontend

React frontend for EduPlatform.

---

## Setup & Run

```bash
Terminal 1 — Backend:
Backend runs on → http://localhost:5000

cd e:\Learning_hub\EduplateForm\eduplatform-backend
npm install
node server.js


Terminal 2 — Frontend:
cd e:\Learning_hub\EduplateForm\eduplatform-frontend
npm install
npm start
Frontend runs on → http://localhost:3000

```

Runs on → http://localhost:3000
